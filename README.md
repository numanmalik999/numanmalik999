## 🚀 Featured Projects

### 🏥 [StudyPrometric](https://www.studyprometric.com)
*A comprehensive preparation platform designed to help medical professionals successfully pass Gulf licensing exams (DHA, SMLE, MOH, HAAD, OMSB, NHRA, and QCHP).*

* **Dynamic Mock Exams**: Fully customizable prep system spanning specialties for Doctors, Dentists, Nurses, Pharmacists, and Physiotherapists.
* **Offline Companion App**: Integrated Android APK companion enabling users to sync progress and study exam questions offline.
* **Automated Daily MCQ Pipeline**: Background cron engines that select a "Question of the Day" and broadcast it to marketing lists, Telegram channels, and user dashboards.
* **Premium Email Delivery**: Implemented privacy-focused BCC batching strategies via Resend API to handle thousands of daily notifications on the free tier.
* **High-Performance Architecture**: Built with Astro for blazing-fast SEO performance and dynamic React SPAs for high-fidelity private user areas.

### 💊 [Pharmasys](https://www.pharmasys.site)
*A robust, clinical-grade Pharmacy Management System designed to optimize pharmaceutical workflows, stock control, and clinical checkups.*

* **Smart Inventory Management**: Real-time batch and expiry tracking, predictive stock reordering, and automated SKU classification.
* **Point of Sale (POS) & Billing**: Sleek cashier interface with digital invoicing, multi-method payment support, and daily financial auditing.
* **Clinical Safety & Drug Interactions**: Integrated checks warning pharmacists of potential drug-to-drug interactions and dosage threshold crossings.
* **Comprehensive Analytics**: Dashboard tracking sales trends, margins, inventory turn rates, and staff performance metrics.
